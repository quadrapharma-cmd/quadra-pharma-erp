# Quadra Pharma ERP

نظام إدارة المصنع الدوائي — ملف HTML واحد لكل مرحلة.

## كيفية الاستخدام
افتح أي ملف مباشرة في المتصفح (Chrome أو Edge) — لا يحتاج سيرفر.

## المراحل

| المرحلة | الملف | الحالة |
|---------|-------|--------|
| 1 — الشركات والمستخدمين | `phase1-companies.html` | ✅ مكتملة |
| 2 — المواد الخام | `phase2-materials.html` | 🔜 قريباً |
| 3 — المنتجات والـ BOM | `phase3-products.html` | 🔜 قريباً |
| 4 — الإنتاج وسجلات الدفعات | `phase4-production.html` | 🔜 قريباً |
| 5 — ضبط الجودة QC | `phase5-quality.html` | 🔜 قريباً |
| 6 — المخزون والشراء | `phase6-inventory.html` | 🔜 قريباً |

## قواعد الكود
- كل مرحلة ملف مستقل — عطل في مرحلة لا يأثر على التانية
- البيانات محفوظة في IndexedDB (غير محدودة الحجم)
- البيانات مشتركة بين كل الملفات (نفس قاعدة البيانات `pharma_erp`)

## للمراجعة والتعديل
افتح issue أو راسل صاحب المشروع بتعديلاتك على كل مرحلة.
